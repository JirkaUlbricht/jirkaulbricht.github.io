About

This project was started on the 10.3.2023 out of mainly boredom and a small challenge to test if I can even accomplish something like this. It isn't anything cool or comparable with any other cloud services regarding matters like security or file storage. It's main purpose is to be convenient to the user and be easily usable. I hope you will enjoy the using Cloud project as I have enjoyed making it all work nice and smooth.

What do I offer?

As stated, the main thing I offer is convenience and easy access to their files whenever and wherever you need. The Cloud project is designed for only holding a smaller volume of files, so it's perfect for stuff like transferring files from different devices, storing your school assignments, having a backup of your files on hand and much more!
